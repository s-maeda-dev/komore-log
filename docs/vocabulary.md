# 💻 開発用語・単語帳

## 場所と方向
- **ローカル**: 自分のPCのこと
- **リモート**: GitHubなどのネット側
- **リポジトリ**: プロジェクトの保管場所
- **クローン**: リモートから自分のPCへコピー
- **コミット**: 自分のPC内での「セーブ」
- **プッシュ**: 自分のPCの記録をネット（GitHub）へ送る
- **プル**: ネットの最新情報を自分のPCに取り込む

## VS Code 拡張機能
- **Extension Pack for Java**: Javaの先生。基本セット。
- **Spring Boot Extension Pack**: Springの助手。入力補助と起動ボタン。
- **GitLens**: コードの履歴書。誰がいつ書いたか確認できる。

## プロジェクトの部品（依存関係）
- **Spring Web**: Webサイトを作る心臓部
- **Thymeleaf**: 画面作成ツール
- **Lombok**: コード自動生成ツール
- **DevTools**: 修正を即座に反映するツール
