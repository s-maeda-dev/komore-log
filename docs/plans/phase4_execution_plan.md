# 📋 Phase 4 実行計画書：フロントエンド実装と UI 刷新（Thymeleaf & Tailwind CSS）

このドキュメントは、`komore-log` のユーザー体験を向上させるため、フロントエンドを整備し、「木漏れ日の静けさ」を感じさせるプレミアムなデザインを適用する手順をまとめたものです。

---

## 1. 概要
現在は最小限のスタイルで動作していますが、これを Tailwind CSS を活用してモダンかつ癒やしを感じるデザインに刷新します。AIプロンプト上の名称を「komore」で統一し、葉っぱをモチーフにしたビジュアルと独自のイラストを導入することで、アプリの世界観を確立します。

---

## 2. 詳細手順

### 4.1 UI 基盤の整備
Tailwind CSS を導入し、共通のレイアウト構造を定義します。

- [x] **Tailwind CSS の導入**
    - CDN を利用したクイックな導入、または必要に応じてカスタムビルドの検討。
- [x] **カラーパレットとタイポグラフィの定義**
    - 「木漏れ日」や「新緑」をイメージした、緑（セージグリーン、フォレストグリーン）・白・ベージュを基調とした配色。
    - 読みやすく落ち着いたフォントの設定。
- [x] **AIキャラクター「komore」のビジュアル導入**
    - 葉っぱをモチーフにした「komore」のアバターイラストを画面上部や返信エリアに配置。

### 4.2 レイアウトとコンポーネントの実装
画面の各要素を Tailwind CSS でスタイリングします。

- [x] **共通レイアウトの作成**
    - ヘッダー、フッター、メインコンテンツエリアの配置。
    - 中央寄せのクリーンなタイムライン表示。
- [x] **投稿フォームの刷新**
    - 入力しやすく、かつ画面に馴染むフローティング風のフォーム。
- [x] **タイムライン（投稿カード）のデザイン**
    - 「グラスモーフィズム（透過ガラス風）」を取り入れたカードデザイン。
    - ユーザーの投稿と AI の返信の視覚的な区別。

### 4.3 ユーザーインタラクションの向上
「AI が考えている時間」や「お返事が届く瞬間」の心地よさを追求します。

- [x] **アニメーションの追加**
    - AI 返信が届いた際のふわっとした表示（フェードイン）。
    - 「考え中...」状態のインジケーターのブラッシュアップ。
- [x] **レスポンシブ対応**
    - スマートフォンでもPCでも、等しく癒やされるレイアウトの調整。

### 4.4 削除機能の UI 統合
- [x] **削除ボタンの設置**
    - カード内にさりげなく配置し、手軽に整理できるようにする。
    - 削除時の確認ダイアログまたはアニメーションによるフィードバック。

## 3. デザインコンセプト
- **キーワード**: 静寂、温もり、透明感、自然
- **メインカラー**: `#f0f4f8` (背景), `#2d5a27` (落ち着いた緑), `#ffffff` (カード)
- **スタイル**: 余白を贅沢に使い、視覚的なノイズを最小限に抑える。
