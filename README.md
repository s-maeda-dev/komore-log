# komore-log
心に寄り添うつぶやきアプリ「komore（こもれ）」のMVP開発。Java/Spring BootとOpenAI APIを活用し、ユーザーの孤独を和らげる最小機能を実装。

# 🌳 komore (こもれ)
**「心に寄り添う、AI共感型つぶやきアプリ」**

誰にも言えない、まとまらない今の気持ちをそっと吐き出すための場所です。
SNSで発信するほどではないけれど、誰かに受け止めてほしい。
そんな瞬間に、AIが木漏れ日のような穏やかな返信を届けます。

## ✨ MVP機能 (Minimum Viable Product)
- **投稿機能**: 今の気持ちをテキスト入力して送信
- **AI共感返信**: OpenAI APIを活用し、投稿に寄り添うメッセージを生成
- **つぶやきログ**: 過去の投稿とAIの返信を時系列で保存・閲覧

## 🛠️ 技術スタック
- **Backend**: Java 21 / Spring Boot 3.x
- **Build Tool**: Maven
- **Database**: MySQL
- **AI**: OpenAI API (GPT-4o)
- **IDE**: Windsurf
